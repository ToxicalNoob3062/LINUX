to compile and install soft:
[!remember its better to see the read me if you want to install any software , the common way is mentioned below.!!]
1)download zip
2)extract and cd into it
3)configure with --prefix=(instalation dir)
4)run (make) command to compile it by the help of Makefile generated by prevstep.
5)sudo make install
6)delete the src file after installation because you dont need it.
7)cd into (instalation-dir/sbin/)
8)run the binary executable most similar to (./soft)
